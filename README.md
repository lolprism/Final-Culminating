## Culminating Project

Game: Capybara Chaos

Capybara Chaos is a survival game developed using Love2D. The player controls a capybara and attempts to survive internet fame by collecting oranges while avoiding increasingly dangerous enemies.

### Controls
- W = Move Up
- A = Move Left
- S = Move Down
- D = Move Right
- Space = Start Game
